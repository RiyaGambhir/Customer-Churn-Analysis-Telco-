# Customer-Churn-Analysis(Telco) - 

## Overview

Analyzed customer churn patterns using SQL, Excel, and Power BI to identify high-risk customer segments and support retention strategies.

## Tools Used

- SQL
- Excel
- Power BI

## Dataset

Telco Customer Churn Dataset from Kaggle.

## Business Questions

- What is the overall churn rate?
- Which contract types have the highest churn?
- How does tenure affect churn?
- What customer segments are most at risk?

## Dashboard Features

- Churn KPI Cards
- Contract Type Analysis
- Tenure Analysis
- Customer Segmentation
- Drill-through Reports
- Pricing Tier

## Recommendations

- Target month-to-month customers.
- Improve onboarding for new customers.
- Offer incentives for longer contracts.
